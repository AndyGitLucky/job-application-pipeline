# Pipeline

Hier entsteht die Sammel- und Aufbereitungsschicht fuer den Market Explorer.

Geplante Teilbereiche:

- Quellen-Adapter
- Job-Normalisierung
- Dedupe
- Analyse-Annotationen
- Aggregation fuer Visualisierungen

Der Fokus hier ist nicht der persoenliche Bewerbungsfluss, sondern ein breiter, analyseorientierter Markt-Datensatz.
